\# AI-Powered SEO Content Production Playbook



\## Project Overview

This repository contains the foundational research and data collection required to build a modern, high-signal AI SEO playbook. The goal of this phase was to identify and analyze top-tier industry practitioners who actively execute and test AI-driven search strategies, rather than those who simply theorize.



\## Source Selection Strategy

The focus of this collection is strictly on \*\*signal over noise\*\*. Instead of scraping 50 generic marketers, I targeted 10 deep-in-the-trenches experts. These individuals were chosen because they:



1\. Conduct live, data-driven tests on AI content performance.

2\. Analyze technical Google algorithm updates and backend AI systems (e.g., SGE/AI Overviews).

3\. Build programmatic or entity-based SEO architecture using LLMs.



\## Repository Structure

\* `/research/sources.md`: The annotated master list of the 10 selected experts.

\* `/research/linkedin-posts/`: High-value posts from experts about AI workflows.

\* `/research/youtube-transcripts/`: Extracted transcripts from SEO videos.

\* `/api\_collection\_script.py`: Python script to collect YouTube transcripts.



\## Methodology

To gather this data, a hybrid approach was used:



\* \*\*YouTube Data:\*\* Extracted using youtube-transcript-api.

\* \*\*LinkedIn Data:\*\* Manually collected high-quality expert posts.
